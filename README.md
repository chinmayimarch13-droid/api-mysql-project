# api-mysql-project
# API + MySQL Integration Project

## Project Overview
This project demonstrates real-world backend integration using:
- Python Flask API
- MySQL Database
- Postman API Testing

## Features
- User Registration API
- MySQL Database Connection
- SQL Data Insertion
- API Testing using Postman
- Backend Validation

## Technologies Used
- Python
- Flask
- MySQL
- Postman

## API Endpoint
POST /register
## URL 
url link for API POSTMAN:http://127.0.0.1:5000/register
## Sample JSON Request

{
  "name":"Chinnu",
  "email":"chinnu@gmail.com"
}

## Expected Response

{
  "message":"User Registered Successfully"
}

## Database Validation Query

SELECT * FROM users;
